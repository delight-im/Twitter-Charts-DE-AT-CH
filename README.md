# Twitter Charts DE-AT-CH

This is a list of the 6,871 most popular Twitter users from Germany, Austria and Switzerland as of May 6, 2014. It is probably the most comprehensive (inofficial) Twitter charts for Germany, Austria and Switzerland.

With a few exceptions, these inofficial Twitter charts include all accounts from Germany, Austria and Switzerland that ever had more than 900 `fans` between April 17, 2013 and May 6, 2014.

`Fans` is our metric for the popularity of a Twitter account and is simply `Followers - Following`.

This page does also include historical records per user for the time between April 17, 2013 and May 6, 2014 in the folder `History`.
